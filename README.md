# Backend for placemi.com

* The backend that powers [placemi.com](http://placemi.com)
* Built using Java and Spring.

## Installation

Ensure that Java 8+ and maven are both installed.

Finally, install all development dependencies:

```
cd CLONED_REPO_DIR
mvn clean install
```
